# Myborg Ponies

**A Kaleidoscopic Rainbow Dystopia for Girls and Boys of all ages!**

*"Rainbow Brite meets Blade Runner. Strawberry Shortcake meets Minority Report. Care Bears meets Mad Max."*

## About

MYBORG PONIES is an interactive website for a fictional world populated by cyborg ponies. The aesthetic blends 1980s girl toy pop art with sci-fi dystopia, featuring pastel colors, neon glows, and dark cyberpunk undertones.

**Tagline:** *Friendship Is Firmware*

## Features

- Character gallery with 12 unique cyborg ponies
- Rich lore and world-building (The Sweet, The Corrupt, The Glitched)
- Fan fiction / data logs with full story reader
- Story submission system via Formspree
- Merchandise showcase
- Scene gallery
- Easter eggs (Konami code, type "pony", Ctrl+Shift+P, triple-click logo)
- PWA support with offline caching
- Responsive design (mobile-first)
- CRT scanline and noise overlay effects
- Animated neon starfield background

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Bungee, Orbitron, Quicksand)
- Formspree (form submissions)
- CounterAPI (visitor counter)
- Service Worker (offline support)
- No frameworks, no build step required

## Deployment

Deployed via GitHub Pages as a static site.
